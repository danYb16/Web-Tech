# Web Tech

## 1. Ce este o resursa?

O resursa este o entitate accesibila printr-un URI, de exemplu o tema, un utilizator sau un produs.

In aplicatia mea, o resursa poate fi o tema accesata printr-un URL precum /teme/1.

---

## 2. Exemplu de URI

https://example.com/teme/1?sort=desc

Componente:
- https → protocol
- example.com → domeniu
- /teme/1 → path (resursa)
- sort=desc → parametru (query)

---

## 3. Parti statice si dinamice

Statice:
- HTML
- CSS
- imagini

Dinamice:
- date din baza de date
- parametri din URL (ex: id)

---

## 4. Tipul aplicatiei

Aplicatia este interactive deoarece utilizatorul poate interactiona cu datele. Poate avea si parti document-centric (pagini statice).
